# Egmon

Egmon est un espace de travail personnel pour centraliser projets, ressources et outils de développement. L'objectif n'est pas de remplacer un outil de gestion de projet complet, mais d'offrir un **second cerveau** léger : savoir sur quoi vous travaillez, où vous en êtes, et retrouver rapidement ce qui compte.

## À propos

Application web pensée pour un usage **personnel** : suivi de projets clients ou persos, tâches, templates réutilisables, liens utiles, notes techniques et idées d'applications — le tout au même endroit.

## Fonctionnalités

- Dashboard avec vue d'ensemble
- Gestion des projets (statut, client, revenus, journal, échéances)
- Tâches en kanban
- Bibliothèque de templates
- Ressources et notes techniques
- Idées d'applications

## Stack

- **Frontend** — HTML5, Tailwind CSS, JavaScript (Vanilla)
- **Base de données & auth** — Supabase
- **Hébergement** — GitHub Pages
